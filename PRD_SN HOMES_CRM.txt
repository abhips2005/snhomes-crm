Build a production-ready Progressive Web App (PWA) called "S N Homes".

Goal:
S N Homes is NOT a normal real estate listing website.
It acts as a real estate agency / property matchmaking platform for Kerala.

Users should only submit simple forms.
All operations happen inside an internal admin CRM.

Tech Stack:
- Frontend: Next.js 15 (App Router, TypeScript)
- Styling: Tailwind CSS
- Backend: Supabase
- Database: Supabase
- ORM: Supabase
- Auth: Admin login only initially
- Storage: Supabase
- State: TanStack Query
- Forms: React Hook Form + Zod
- Notifications: WhatsApp integration placeholder
- Deployable via Docker (Also in vercel)
- PWA enabled (installable)

Architecture:
Frontend:
apps/web

Backend:
apps/api

Shared:
packages/types

Goal:
Build V1 only.
